## 🧱 Сверточные блоки

- **Residual Block (ResNet)** — два свёрточных слоя 3×3 с skip-соединением, суммирующим вход и выход для облегчения обучения глубоких сетей.
- **Bottleneck Residual Block** — облегчённый вариант с тремя свёртками (1×1, 3×3, 1×1) для уменьшения и восстановления размерности.
- **Inception Module** — параллельные свёртки разного размера (1×1, 3×3, 5×5), объединяемые конкатенацией.
- **Dense Block** — каждый слой соединён со всеми последующими, выходы конкатенируются, улучшая переиспользование признаков.
- **ResNeXt Block** — группировка каналов на несколько параллельных ветвей (кардинальность) для повышения эффективности.
- **Fire Module** — легковесная конструкция из «сжимающего» слоя 1×1 и «расширяющей» части из параллельных 1×1 и 3×3 свёрток.
- **Non-Local Block** — фиксирует дальние пространственно-временные зависимости через взвешенное суммирование признаков со всего входа.
- **Depthwise Separable Convolution** — раскладывает обычную свёртку на глубинную (пространственную) и поточечную (канальную) для резкого сокращения вычислений.
- **Atrous (Dilated) Convolution** — свёртка с «дырками», увеличивающая поле обзора без потери разрешения.

## 🎯 Блоки внимания (Channel / Spatial)

- **Squeeze-and-Excitation (SE) Block** — канальное внимание: глобальный пулинг → MLP → сигмоида → взвешивание каналов.
- **Spatial Attention Module** — генерирует карту внимания по пространству (где смотреть), используя средний и максимальный пулинг.
- **Channel Attention Module** — аналог SE, но веса каналов вычисляются с использованием и среднего, и максимального отклика.
- **Axial Attention** — эффективное self-attention для изображений: сначала внимание по строкам, затем по столбцам.

## ⚙️ Основные блоки Transformer

- **Multi-Head Self-Attention (MHA)** — параллельное внимание в нескольких проекциях для разностороннего анализа последовательности.
- **Feed-Forward Network (FFN)** — два линейных слоя с активацией, применяемых позиционно независимо.
- **Layer Normalization (LayerNorm)** — нормализация по признакам для каждого токена, стабилизирующая обучение.
- **Residual Connection** — суммирование входа и выхода подслоя для градиентного потока.
- **Dropout** — случайное обнуление нейронов во время обучения для регуляризации.
- **Positional Encoding** — добавление информации о порядке элементов в последовательность.
- **Cross-Attention** — позволяет декодеру учитывать выходы кодера при генерации.

## 🔁 Рекуррентные блоки

- **LSTM (Long Short-Term Memory)** — ячейка с вентилями входа, забывания, выхода и долговременной памятью.
- **GRU (Gated Recurrent Unit)** — упрощённый вариант LSTM с двумя вентилями (сброс и обновление), более быстрый и лёгкий.

## ✨ Продвинутые механизмы внимания

- **Multi-Query Attention (MQA)** — все головы разделяют ключи и значения, но имеют свои запросы. Сокращает KV-кеш.
- **Grouped-Query Attention (GQA)** — головы разбиваются на группы, внутри группы общие ключи и значения.
- **FlashAttention** — алгоритм точного внимания, оптимизированный под иерархию памяти GPU для длинных последовательностей.
- **Linear (Kernelized) Attention** — семейство методов, заменяющих softmax линейной функцией для достижения сложности O(N).
- **Sparse Attention** — вычисление внимания только по заданному паттерну (локальное, глобальное, случайное).
- **Sliding Window Attention** — каждый токен видит лишь соседей в фиксированном окне. Базовый кирпич Mistral.
- **Deformable Attention** — ключи/значения берутся лишь с малого числа обучаемых смещённых позиций, экономя вычисления.
- **Attention on Attention (AoA)** — двухступенчатое внимание с дополнительным gate для фильтрации релевантной информации.
- **Cross-Attention with Memory** — запросы взаимодействуют с фиксированной памятью из нескольких закодированных представлений.
- **Graph Attention (GAT)** — self-attention для графов: веса вычисляются динамически только между соседними вершинами.

## 🌍 Механизмы World Models

- **Perception Module (Encoder)** — сжимает сенсорные данные (изображения) в компактное латентное представление (часто VAE).
- **Recurrent State-Space Model (RSSM)** — объединение детерминированной (GRU) и стохастической скрытых переменных; ядро семейства Dreamer.
- **Dynamics (Transition) Model** — предсказывает следующее латентное состояние по текущему и действию.
- **Reward Model** — предсказывает награду по латентному состоянию, позволяя «грезить» о будущем.
- **Continue (Terminal) Model** — оценивает вероятность окончания эпизода в данном латентном состоянии.
- **World Transformer** — замена рекуррентной модели мира на Transformer, работающий с историей латентов и действий.
- **Latent Overshooting** — техника обучения с разворачиванием динамики на несколько шагов для стабилизации долгосрочных предсказаний.
- **Joint Embedding Predictive Architecture (JEPA)** — предсказание не пикселей, а эмбеддингов одного фрагмента по контексту другого (энергоэффективная модель мира).
- **Mixture Density Network (MDN) Head** — выходная голова моделирует смесь распределений (гауссиан) для мультимодальных переходов.
- **Imagination Rollout** — процесс генерации воображаемой траектории в латентном пространстве с помощью dynamics и reward моделей.
